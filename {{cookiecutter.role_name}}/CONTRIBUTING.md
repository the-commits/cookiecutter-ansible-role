# Contributing

This repository is a **read-only mirror** of the primary repository on
[sourcehut](https://git.sr.ht).

**Do not open pull requests or issues here.** All contributions are managed
through sourcehut:

- **Repository:** https://{{ cookiecutter.git_host }}/~{{ cookiecutter.git_user }}/{{ cookiecutter.role_name }}
- **Patches:** send via email to `~{{ cookiecutter.git_user }}/{{ cookiecutter.role_name }}@lists.sr.ht`
- **Issues:** https://todo.sr.ht/~{{ cookiecutter.git_user }}/{{ cookiecutter.role_name }}

The GitHub mirror is updated automatically on every push to sourcehut.
